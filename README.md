# Zenith

Zenith — это постиндустриальный survival-движок и игра, построенные на Java 21 и LWJGL 3 (OpenGL 3.3). Проект ориентирован на **Data-Driven Development**, где логика отделена от данных, а взаимодействие с миром строится на физике и процедурных эффектах.

## Технологический стек
- **Core:** Java 21 (Record, Pattern Matching).
- **Graphics:** LWJGL 3 (OpenGL 3.3+ Core Profile).
- **Math:** JOML.
- **Networking:** Kryonet (UDP/TCP).
- **Build System:** Gradle.

## Ключевые системы

### 1. Движок и Рендеринг
- **Identifier-First Dynamic ID System**: Полный отказ от ручного управления числовыми ID в JSON. Реестры автоматически назначают индексы на основе строковых `Identifier`.
- **Texture Array Engine (v4.0)**: Использование `GL_TEXTURE_2D_ARRAY` для устранения мерцания швов текстур и корректной работы мип-мапов.
- **Skeletal Viewmodel System**: Иерархическая система костей (Shoulder -> Forearm -> Hand) для плавной и реалистичной анимации от первого лица.
- **Unified Shader Pipeline**: Унифицированное декодирование свойств блоков (`decodeBlockInfo`) в шейдерах для поддержки Connected Textures и биомного тинта независимо от ID.

### 2. Игровые механики (Survival)
- **Hytale-style Block Breaking**: Система "слабых точек" (Weak Spots) и процедурных шрамов (Persistent Scars) на блоках. Индикация прогресса через раскаление инструмента (Mining Heat VFX).
- **In-World Crafting**: Крафт непосредственно на поверхностях блоков (Пень, Стол мусорщика) без использования модальных окон инвентаря.
- **Scrap Metal Era**: Прогрессия через эксгумацию ресурсов из руин (разбор машин фомкой, сборка инструментов на заклепках и изоленте).
- **Noise & AI System**: Глобальная система шума. Мобы (Скауты) реагируют на звуки действий игрока, переходя из режима поиска в преследование.

### 3. Интерфейс 
- **Data-Driven GUI**: Полная конфигурация окон через JSON (якоря, юниты `1s`, адаптивные фоны).
- **Advanced UI Components**: Переиспользуемые компоненты вроде `UISearchBar` с поддержкой выделения, буфера обмена и системного ввода.
- **Survivor's Tablet**: Внутриигровой справочник (Дневник) с динамической загрузкой категорий и рецептов.

### 4. Инструментарий
- **Animation Studio**: Встроенный 3D-редактор анимаций (FK/IK FABRIK) с экспортом в JSON и поддержкой Auto-Keying.
- **Developer Panel**: Продвинутая панель отладки с умным поиском и сортировкой контента.

## Структура проекта
- `src/main/java/com/za/zenith/engine`: Ядро движка, графика, ввод.
- `src/main/java/com/za/zenith/world`: Логика мира, генерация, блоки, предметы.
- `src/main/resources/zenith`: Игровые данные (JSON), шейдеры, текстуры.
- `.gemini/memory`: Актуальная техническая документация и Single Source of Truth проекта.

## Сборка и запуск
Требуется JDK 21.
```bash
./gradlew run
```
Сборка дистрибутива:
```bash
./gradlew build
```
