# Текущий статус проекта "MinecraftButBetter"

## Реализованные фичи
- **Глубокая интеграция MCP (NEW)**:
  - Подключены и настроены серверы `memory` (Knowledge Graph), `github` и `java-lsp` в глобальном конфиге.
  - Построен начальный архитектурный граф знаний проекта (сущности Core, Graphics, World, Entities, Energy).
  - Обновлены стандарты разработки (`GEMINI.md`) с упором на использование MCP и LLM-friendliness (подробные докстринги, явные связи в графе).
- **Улучшение UX инвентаря**:
  - Реализовано деление стеков правой кнопкой мыши (взять половину / положить один).
  - Добавлены горячие клавиши: `Q` (выбросить один), `Ctrl+Q` (выбросить стек) — работают как в игре, так и в инвентаре.
  - Реализовано выбрасывание предметов при клике вне окон инвентаря.
  - Автоматическое объединение стеков при клике левой кнопкой мыши.
  - **Drag-to-Distribute**: Раскладка предметов «мазком» (ЛКМ — поровну, ПКМ — по одному). Умное выделение слотов.
  - **Quick Sort (клавиша Z)**: Автоматическая сортировка и объединение стаков в инвентаре.
  - **Тултипы**: Добавлено отображение числового ID предмета (полезно для разработки рецептов).
- **Ядро и движок**: Базовый игровой цикл (GameLoop), управление окном через GLFW, рендеринг через OpenGL.
- **Мир**: Генерация бесконечного (в теории) мира по чанкам, сохранение изменений блоков в оперативной памяти.
- **Сущности**: Физика игрока (гравитация, коллизии, прыжки, режим полета).
- **Инструменты и блоки (UPDATED)**:
  - Добавлен **Admin Hammer** (Инструмент разработчика): мгновенно разрушает любые блоки. Использует текстуру `nether_star.png`.
  - Полный набор стандартных блоков (камень, земля, трава, дерево и т.д.).
  - Специальные блоки: Ступеньки (Stairs) и Полублоки (Slabs).
  - Механика "Вертикального режима" (клавиша R) для установки полублоков в вертикальном положении.
- **Система предметов**:
  - Абстракция `Item` и `ItemStack` для поддержки любых предметов в инвентаре.
  - Базовые инструменты: Каменный нож, Кирка из арматуры, Ломик, Каменный топор.
  - Реестр предметов `ItemRegistry` для легкого добавления новых типов.
- **Механика добычи и прочности**:
  - Прогрессивное разрушение блоков (не мгновенное).
  - Эффективность инструментов зависит от типа блока.
  - Износ инструментов при использовании.
- **Система выживания**:
  - Механика голода (`Hunger`) и насыщения (`Saturation`).
  - Предметы еды (Raw Meat, Cooked Meat, Canned Food) с ванильными иконками.
  - Механика Костра (Campfire).
- **Система Шума и Стелса (UPDATED)**:
  - Механика "Пола шума" (Floor Noise): шум моментально принимает базовое значение действия (ходьба, бег, ломание) и затухает от него.
  - Поддержка скрытности (Sneaking, клавиша Shift): уменьшенная скорость передвижения и минимальный уровень шума.
  - Накопительный эффект (Additive Spikes): прыжки и разрушение блоков добавляют шум поверх базового уровня.
  - Исправлен баг "тихой добычи": теперь ломание блоков корректно генерирует шум даже на месте.
  - Визуальный индикатор шума в HUD (динамический цвет от зеленого к красному).
- **Инвентарь и GUI (v2.0):**
    - Реализована полноценная сетка инвентаря 9x4 (27 основных слотов + 9 хотбар).
    - Система Drag-and-Drop: перекладывание предметов между слотами мышкой.
    - Визуализация: тултипы при наведении, отображение количества в стеке и прочности инструментов.
    - Управление: Открытие/закрытие на клавишу `E` (и ESC), освобождение курсора при открытом инвентаре.
- **Интерфейс**:
  - Хотбар с поддержкой отрисовки блоков и инструментов (ванильные иконки).
  - Отображение названий предметов (локализованных через `I18n`) и прогресса разрушения блока (%).
  - Локализованы системные сообщения (Голод, Шум, Добыча) и меню паузы.
  - Исправлена ориентация текстур UI (не перевернутые).
- **Игровые режимы и Панель Разработчика (NEW)**:
  - Реализован фундамент переключения режимов: `SURVIVAL` (выживание) и `DEVELOPER` (креатив).
  - Переключение режимов на клавишу `F3`.
  - Реализована панель разработчика (Creative Inventory) в GUI: сетка 4x12 справа от основного инвентаря.
  - Поддержка скролла в панели разработчика через колесико мыши.
  - Возможность бесконечного получения любых зарегистрированных блоков и предметов из панели в инвентарь игрока.
- **Локализация и шрифты (UPDATED)**:
  - Реализована система `I18n` с поддержкой JSON-файлов (`ru_ru.json`, `en_us.json`).
  - Исправлен `FontRenderer`: реализована корректная обработка Unicode-суррогатных пар при парсинге `default.json` и отрисовке строк.
  - Поддержка кириллицы через `nonlatin_european.png` и других символов через динамическую подгрузку Unicode-страниц (`unicode_page_*.png`).
  - Автоматический расчет ширины глифов на основе прозрачности пикселей для всех типов провайдеров (bitmap, legacy_unicode).
  - Внедрен кэш текстур для оптимизации памяти и производительности.
- **Система Сущностей и ИИ (UPDATED)**:
  - Реализована иерархия сущностей: `Entity` -> `LivingEntity` -> `Player` / `ScoutEntity`.
  - Введена физика для всех сущностей (гравитация, коллизии с миром).
  - Реализован ИИ "Скаутов" (зараженных), реагирующий на шум игрока.
  - Состояния ИИ: Блуждание (Wander), Поиск по звуку (Search), Погоня (Chase).
  - Реализована логика "забывания" игрока: после потери цели моб ищет её 7 секунд.
  - Визуализация состояния моба (изменение цвета в зависимости от уровня агрессии).
- **Механика использования предметов (UPDATED)**:
  - Еда теперь употребляется в любом состоянии (включая прыжок или взгляд в небо).
  - Приоритезация действий ПКМ (Interactive > Use > Place).
- **Энергосистема и Индустрия (UPDATED)**:
  - Реализована инфраструктура `BlockEntity` для блоков со сложной логикой и состоянием.
  - Добавлен интерфейс **IEnergyStorage** — единый стандарт передачи и хранения энергии.
  - Добавлен **Бензогенератор** (Generator): вырабатывает энергию из топлива.
  - Реализованы **Электросети** (Cables): блоки-проводники, распределяющие энергию между узлами.
  - Добавлен первый потребитель — **Электрическая лампа** (Electric Lamp).
  - Добавлен **Аккумулятор** (Battery): пассивное хранилище на 10 000 единиц энергии для сглаживания пиков потребления.
  - Система "Глобального шума": мобы теперь слышат не только игрока, но и работающую технику.
- **First-Person Mechanics & Item Entities (v2.0)**:
  - Визуализация руки игрока и удерживаемого предмета (View Model).
  - Автоматическое переключение режима отображения: **3D блоки** и **3D модели предметов** (с толщиной) через `ItemMeshGenerator`.
  - Улучшенное позиционирование рук и наклон для более естественного вида.
  - **Система выпадающих предметов (v2.1 - Polished):**
    - Предметы больше не являются "белыми кубами": блоки рендерятся как миниатюрные блоки, предметы — как 3D модели (с толщиной).
    - **Реалистичная физика:** исправлено направление броска (теперь точно по взгляду), дугообразная траектория.
    - **Уникальная механика веса:** тяжелые предметы (блоки) имеют в 2.5 раза больший вес, что влияет на скорость падения, трение о землю и силу броска.
    - **Корректное вращение:** исправлен центр вращения (pivot) для блоков и предметов. Инструменты теперь вращаются вокруг своей оси, сохраняя биллборд-ориентацию.
  - Подбор предметов игроком при сближении.
  - Анимации взаимодействия: плавная инерция при движении (View Bobbing), сочная анимация удара (Swing).
  - **Рендеринг View Model (предмет в руке):**
    - Внедрена система динамических трансформаций через объекты `Item`.
    - **TODO (Для другого разработчика):** Реализовать систему отображения рук.
        - Текущая реализация рук удалена для полной переработки.
        - Требуется: поддержка скинов, скелетная привязка к предмету, синхронизированные анимации и нормальное расположение предметов в руке.
    - Исправлена намотка (winding order) и нормали мешей.
    - Внедрен фиксированный FOV (70 градусов) для View Model.
    - Добавлена поддержка затенения через камерно-зависимый свет.
- **Tier 1: Примитивное выживание (In Progress)**:
  - **Foraging**: Механика сбора палок и камней с земли (GroundItemBlock).
  - **Napping**: Интерфейс 5x5 для создания лезвий инструментов путем откалывания кусков камня.
  - **Сборка на пне (In-World Crafting)**: Создание инструментов прямо в мире путем размещения компонентов на срезе бревна (Пень - STUMP).
  - **Recipe System**: Универсальный реестр рецептов, загружающий JSON-конфиги по числовым ID.

## Последние изменения (Исправления и Оптимизация)
- **Устранение критических утечек памяти и зависаний**: 
  - Исправлен бесконечный цикл каскадного обновления чанков при генерации лесов (`World.setBlock` с флагом `notify=false`).
  - Внедрено жесткое кэширование мешей предметов и блоков в `Renderer.java`, предотвращающее переполнение видеопамяти.
  - Отменен спавн тысяч `BlockEntity` для каждого дерева. Теперь `StumpBlockEntity` создается лениво, только при взаимодействии с блоком (превращение WOOD в STUMP).
- **Исправлена плавность игры**: Устранены рывки камеры при открытии инвентаря (синхронизация перенесена в конец физического шага).
- **Удалены фризы**: Избавление от блокирующих циклов `while` при нажатии клавиш ESC и E. Теперь ввод неблокирующий.
- **Исправление Рендеринга и CTM**:
  - Исправлен критический баг в `Renderer.java`, где в шейдер передавались неверные UV-координаты (использовался `minV` вместо `maxV`). Это исправляет "бежевое стекло", поломку Connected Textures и отсутствие окрашивания травы/листвы.
  - Оптимизирован `ChunkMeshGenerator.java`: возвращено исключение отрисовки внутренних граней для стекла (как в стабильной версии), что снижает количество вершин без потери CTM-эффектов.
- **Исправление UI и Голода**:
  - Исправлен баг отображения количества предметов в хотбаре.
  - Исправлен критический баг отрисовки голода: при значении 0 больше не выводится `%.1f` вместо числа.
- **Исправление Генератора**:
  - Добавлена информация о состоянии (топливо, энергия) при взаимодействии с генератором.

## Roadmap (Глобальная индустриализация)

Прогрессия разделена на 5 технологических эпох (Tiers).

### Tier 1: Примитивное выживание (Scavenging)
- **СТАТУС: ВОССТАНОВЛЕНИЕ ПОСЛЕ ОТКАТА**
- [ ] Сбор ресурсов с земли (Foraging).
- [ ] Механика скалывания (Napping).
- [ ] Базовая система рецептов через JSON.
- [ ] Огонь и обжиг глины (Pit Kiln).

### Milestone: Guide & Knowledge (Tier 1.5)
- **СТАТУС: ПЛАН**
- Система контекстных подсказок (Tutorial).
- "Дневник выжившего" (In-game Guide) с чертежами.

### Tier 2: Ранняя Механика (Kinetic Age)
- **СТАТУС: ПЛАН**
- Кинетическая энергия (валы, шестерни, водяные колеса).
- Жернова (Millstone) для дробления руды.
- Механический молот для ковки пластин.
- Сплавы (Бронза).

### Tier 3: Паровая эпоха (Steam Age)
- **СТАТУС: ПЛАН**
- Бойлеры, пар под давлением, риск взрыва.
- Многоблочные структуры (Доменная печь).
- Базовая логистика (трубы, конвейеры).

### Tier 4: Электрификация (Electric Age)
- **СТАТУС: ЧАСТИЧНО ГОТОВО** (Генератор, кабели, лампа, аккумулятор)
- Интеграция с единой `EnergyNetwork`.
- Переработка жидкостей (нефть).
- Сборочные автоматы (Assemblers).

### Tier 5: Оборона и Эндгейм (The Siege)
- **СТАТУС: ПЛАН**
- Радиовышка (цель игры).
- Оборонительные системы (турели, заборы под напряжением).
- Система "Осад" (Siege) — реакция мира на шум и загрязнение.
