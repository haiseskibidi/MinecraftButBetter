# Текущий статус проекта "MinecraftButBetter"

## Реализованные фичи
- **Ядро и движок**: Базовый игровой цикл (GameLoop), управление окном через GLFW, рендеринг через OpenGL.
- **Мир**: Генерация бесконечного (в теории) мира по чанкам, сохранение изменений блоков в оперативной памяти.
- **Сущности**: Физика игрока (гравитация, коллизии, прыжки, режим полета).
- **Инструменты и блоки (UPDATED)**:
  - Добавлен **Admin Hammer** (Инструмент разработчика): мгновенно разрушает любые блоки. Использует текстуру `nether_star.png`.
  - Полный набор стандартных блоков (камень, земля, трава, дерево и т.д.).
  - Специальные блоки: Ступеньки (Stairs) и Полублоки (Slabs).
  - Механика "Вертикального режима" (клавиша R) для установки полублоков в вертикальном положении.
- **Система предметов**:
  - Абстракция `Item` и `ItemStack` для поддержки любых предметов в инвентаре.
  - Базовые инструменты: Каменный нож, Кирка из арматуры, Ломик.
  - Реестр предметов `ItemRegistry` для легкого добавления новых типов.
- **Механика добычи и прочности**:
  - Прогрессивное разрушение блоков (не мгновенное).
  - Эффективность инструментов зависит от типа блока.
  - Износ инструментов при использовании.
- **Система выживания**:
  - Механика голода (`Hunger`) и насыщения (`Saturation`).
  - Предметы еды (Raw Meat, Cooked Meat, Canned Food) с ванильными иконками.
  - Механика Костра (Campfire).
- **Система Шума и Стелса (UPDATED)**:
  - Механика "Пола шума" (Floor Noise): шум моментально принимает базовое значение действия (ходьба, бег, ломание) и затухает от него.
  - Поддержка скрытности (Sneaking, клавиша Shift): уменьшенная скорость передвижения и минимальный уровень шума.
  - Накопительный эффект (Additive Spikes): прыжки и разрушение блоков добавляют шум поверх базового уровня.
  - Исправлен баг "тихой добычи": теперь ломание блоков корректно генерирует шум даже на месте.
  - Визуальный индикатор шума в HUD (динамический цвет от зеленого к красному).
- **Интерфейс**:
  - Хотбар с поддержкой отрисовки блоков и инструментов (ванильные иконки).
  - Отображение названий предметов и прогресса разрушения блока (%).
  - Исправлена ориентация текстур UI (не перевернутые).
- **Система Сущностей и ИИ (UPDATED)**:
  - Реализована иерархия сущностей: `Entity` -> `LivingEntity` -> `Player` / `ScoutEntity`.
  - Введена физика для всех сущностей (гравитация, коллизии с миром).
  - Реализован ИИ "Скаутов" (зараженных), реагирующий на шум игрока.
  - Состояния ИИ: Блуждание (Wander), Поиск по звуку (Search), Погоня (Chase).
  - Реализована логика "забывания" игрока: после потери цели моб ищет её 7 секунд.
  - Визуализация состояния моба (изменение цвета в зависимости от уровня агрессии).
- **Механика использования предметов (UPDATED)**:
  - Еда теперь употребляется в любом состоянии (включая прыжок или взгляд в небо).
  - Приоритезация действий ПКМ (Interactive > Use > Place).
- **Энергосистема и Индустрия (UPDATED)**:
  - Реализована инфраструктура `BlockEntity` для блоков со сложной логикой и состоянием.
  - Добавлен интерфейс **IEnergyStorage** — единый стандарт передачи и хранения энергии.
  - Добавлен **Бензогенератор** (Generator): вырабатывает энергию из топлива.
  - Реализованы **Электросети** (Cables): блоки-проводники, распределяющие энергию между узлами.
  - Добавлен первый потребитель — **Электрическая лампа** (Electric Lamp).
  - Добавлен **Аккумулятор** (Battery): пассивное хранилище на 10 000 единиц энергии для сглаживания пиков потребления.
  - Система "Глобального шума": мобы теперь слышат не только игрока, но и работающую технику.

## Последние изменения (Исправления)
- **Архитектура**: Энергия передается методом "сообщающихся сосудов" — кабели выравнивают заряд между всеми подключенными `IEnergyStorage`.
- Унифицирован поиск соседних блоков через `com.za.minecraft.utils.Direction`.

## В работе
- Верстаки: рабочие станции для создания компонентов электроники.
- Механика "Разбора" (Dismantling) для получения ресурсов.
